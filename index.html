Quality gates that flow

What is slop-mop?

slop-mop is a Python-based quality gate orchestration tool that wraps your linters, type checkers, and tests into a unified workflow.

Instead of running pytest, black, mypy, and flake8 separately — with no caching, no ordering, no remediation guidance — you run sm swab. It:

  • Runs gates in dependency order
  • Caches clean results across commits
  • Auto-fixes what it can
  • Tells you exactly what to do next

The workflow

Edit code → run sm swab → fix issues → push → run sm buff to triage CI → address threads → ship.

Or just run sm sail and let it drive the loop automatically.

Get started

Install from PyPI:

pipx install slopmop[all]
